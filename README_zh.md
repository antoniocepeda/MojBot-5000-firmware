# MojBot 5000 Firmware

该仓库现在是 MojBot 机器人狗固件仓库。

目前这是从早期开发代码中拆分出来的独立仓库，目标是逐步清理旧的上游品牌、文档和无关板级内容，让它真正成为 MojBot 自己的固件项目。

当前重点：
- 保留 MojBot 机器人狗所需的固件能力
- 保留我们自己的配置拉取 / 本地缓存 / 刷机工具链
- 逐步移除与 MojBot 无关的上游历史包袱

相关仓库：
- `MojBot-5000` → 网站 / 设置流程 / 后台 / 产品层
- `MojBot-5000-runtime` → 运行时服务 / WebSocket / 语音 / 指令控制
