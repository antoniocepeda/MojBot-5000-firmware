# MojBot 5000 Firmware

このリポジトリは MojBot ロボットドッグ用ファームウェアの独立リポジトリです。

現時点では、初期の開発コードから切り出した状態であり、今後は古い上流ブランディングや不要なボード向け記述を整理して、MojBot 専用のファームウェアリポジトリにしていきます。

現在の重点：
- MojBot ロボットドッグに必要なファームウェア機能を維持する
- 独自の設定取得 / ローカルキャッシュ / フラッシュツールを維持する
- MojBot と無関係な上流由来の残骸を段階的に削減する

関連リポジトリ：
- `MojBot-5000` → Web サイト / セットアップ導線 / 管理画面 / プロダクト層
- `MojBot-5000-runtime` → ランタイムサービス / WebSocket / 音声 / コマンド制御
